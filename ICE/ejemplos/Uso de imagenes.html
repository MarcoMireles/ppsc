Uso de imagenes

El uso de imagenes es esencial para llamar la atencion del cliente y que sea un poco mas estetico
lo puedes usar de maneras diferentes en las que te puedes imaginar y como el cliente te lo pida
tambien se puede incluso hacer animaciones cuando el cursor pase aparezca la imagen

Para hacer uso de las imagenes se necesita la eqtiqueta "img"
consecuentemente se coloca adentro el link de la imagen que quisieras colocar
y eso seria todo ahora solo seleccionas donde la quisieras colocar y estaria listo
